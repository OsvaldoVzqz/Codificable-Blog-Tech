Conoce sobre blockquote

Si alguna vez has deseado dar énfasis y destacar ideas clave en tu contenido, este es el lugar adecuado para ti. En esta breve exploración, te sumergirás en el fascinante mundo del resaltado efectivo con la habilidad de capturar la atención mediante elementos visuales distintivos. Prepárate para descubrir cómo realzar tus mensajes y añadir un toque de intriga a tus lectores, mientras exploramos la magia de enfocar la mirada en fragmentos esenciales.

¿Qué es Blockquote?

El elemento <blockquote> es una etiqueta de marcado HTML diseñada para representar fragmentos de texto que se citan o extraen de otra fuente. Usualmente, se utiliza para dar crédito a autores y proporcionar un contexto claro para el contenido citado. Además, el elemento <blockquote> es una herramienta valiosa para la semántica web, ya que indica que el contenido dentro de él es una cita.

Sintaxis básica

La sintaxis básica del elemento <blockquote> es la siguiente:

HTML

<blockquote>
        <p>El texto citado va aquí</p>
        <footer>Nombre del autor</footer>
</blockquote>
  • <blockquote>: Inicia el bloque de cita.
  • <p>: Define el párrafo que contiene el texto citado.
  • <footer>: Se usa para indicar el autor o la fuente de la cita.

Atributos clave

Aunque el elemento <blockquote> es relativamente simple, admite algunos atributos clave que pueden mejorar su apariencia y funcionalidad:

  • <cite>: Este atributo se utiliza para proporcionar una URL que enlaza con la fuente original del contenido citado. Si bien no es visible para los usuarios, es importante para la accesibilidad y para dar crédito adecuado a la fuente.

Ejemplos:

<blockquote>
    <p>La imaginación es más importante que el conocimiento.</p>
    <cite>— Albert Einstein</cite>
</blockquote>

    <p>El cambio climático es el mayor desafío al que se enfrenta la humanidad en este siglo.</p>

    </cite>
</blockquote>
